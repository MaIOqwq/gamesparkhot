# Changelog

## v1.0.0 (2024-06-11)

Initial open-source release.

### Added
- Multi-platform crawler framework (Bilibili, Douyin, Weibo, Xiaohongshu, Zhihu, Tieba, Kuaishou, NGA)
- Kafka message queue integration for crawler data
- Spark Streaming data cleaning and normalization pipeline
- Spring Boot REST API backend with Redis caching
- Vue 3 + Vite interactive dashboard with ECharts visualizations
- LSTM and XGBoost models for popularity trend prediction
- PaddleNLP-based sentiment analysis (ERNIE/SKEP)
- Revisit daemon for intelligent re-crawl scheduling
- Author stats tracking and ranking
- Hotspot detection and warning system

### Notes
- First open-source release. Code has been desensitized for public distribution.
- See README.md and docs/ for detailed usage instructions.
- Pretrained model files and large artifacts have been excluded; see models/ for training scripts.
